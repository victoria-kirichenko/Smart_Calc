Smart_Calculator

About programm

The program is implemented in the Qt Creator IDE. The calculator has the ability to calculate basic and trigonometric functions, as well as entering "x".

How to start

To start using the calculator, you need to install it using the make install command. this command will install the program on the desktop in the SmartCalc folder.

The rules

When you enter "x" in the expression and press the "graph" button, a new window will open where you can plot the graph. You can choose the range (X min, X max) and the interval of axes (-X; X) and (-Y; Y) by entering a positive number X and Y for them. Use the "Draw" button to draw a graph. If you press "=" while expressing with "x", you will also be taken to the graph. If you enter an expression incorrectly, you will be asked to enter the expression again.

Unistall

to remove the user program. can use the make uninstall command
